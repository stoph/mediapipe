# Testing out Google's Mediapipe

## Setup and Usage

1. **Create and activate a virtual environment:**

```bash
python3 -m venv venv
source venv/bin/activate
```

2. **Install dependencies:**

```bash
pip install opencv-python mediapipe
```

3. **Run the script:**

```bash
python3 live.py
```
